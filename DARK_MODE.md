# 🌙 深色模式配色方案

## 配色详情

### 主色调
```
背景色：      #1a1a1a  (深黑色)
卡片背景：    #2a2a2a  (深灰色)
边框颜色：    #333333  (中灰色)
```

### 文字颜色
```
主要文字：    #e8e8e8  (接近白色)
次要文字：    #888888  (灰色)
占位符：      #666666  (深灰色)
```

### 强调色
```
主题蓝：      #4a9eff  (亮蓝色)
悬停蓝：      #3a8eef  (深蓝色)
已收藏背景：  #1e3a5f  (深蓝灰)
已收藏悬停：  #24436e  (中蓝灰)
```

### 交互效果
```
卡片悬停：    #303030  (稍亮的深灰)
按钮悬停：    #333333  (中灰)
阴影效果：    rgba(74, 158, 255, 0.25)  (蓝色半透明)
```

### 滚动条
```
轨道背景：    #2a2a2a
滑块颜色：    #444444
滑块悬停：    #555555
```

## 🎨 视觉特点

### 1. **护眼设计**
- 深色背景减少屏幕亮度
- 柔和的蓝色主题色（#4a9eff）不刺眼
- 文字使用 #e8e8e8 而非纯白，降低对比度

### 2. **层次分明**
```
顶层（最亮）：卡片悬停 #303030
中层：       卡片背景 #2a2a2a
底层（最暗）：容器背景 #1a1a1a
```

### 3. **状态清晰**
- 普通卡片：深灰色背景
- 已收藏：蓝灰色背景（#1e3a5f）
- 悬停效果：边框发光 + 轻微上浮

### 4. **自定义滚动条**
- 与深色主题完美融合
- 圆角设计，视觉柔和
- 悬停时颜色变化

## 🔧 自定义配色

如果你想调整颜色，修改脚本中的 CSS 变量：

### 调整背景亮度
```css
.static-favlists-grid {
    background: #1a1a1a;  /* 改成 #0d0d0d 更暗，#252525 更亮 */
}
```

### 调整主题色
```css
.static-favlist-btn.collect {
    background: #4a9eff;  /* 改成你喜欢的颜色 */
}
```

### 调整文字亮度
```css
.static-favlist-name {
    color: #e8e8e8;  /* 改成 #ffffff 更亮，#d0d0d0 更暗 */
}
```

## 💡 使用建议

1. **夜间使用**：深色模式在夜间使用更护眼
2. **长时间阅读**：减少眼睛疲劳
3. **省电模式**：OLED 屏幕可节省电量
4. **专注模式**：深色背景更易专注内容

## ⚙️ 切换浅色模式

如果你想改回浅色模式，只需要把所有颜色反转：

```css
/* 快速切换方案 */
背景：#1a1a1a  →  #ffffff
卡片：#2a2a2a  →  #ffffff
文字：#e8e8e8  →  #1a1a1a
边框：#333333  →  #e1e4e8
按钮：#4a9eff  →  #0066ff
```

或者直接使用脚本的 v1.0.0 版本（浅色模式）。
